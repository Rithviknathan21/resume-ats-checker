# 🚀 Smart Resume ATS Checker

A simple NLP-based tool that analyzes how well a resume matches a job description.

## 💡 Overview
This project simulates an **Applicant Tracking System (ATS)** used by companies to filter resumes based on keywords.

It compares:
- Resume contents
- Job description

And provides:
- 📊 Match percentage
- ❌ Missing keywords
- 💡 Suggestions for improvements

---

## ⚙️ Tech Stack
- Python
- Scikit-learn
- Natural Language Processing (NLP)

---

## 🧠 Features
- Keyword extraction
- Cosine similarity matching
- Resume improvement suggestions
- Simple and fast execution

---

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Paste your resume text
3. Paste job description
4. Get instant analysis

---

## 📂 Project Structure
- ats_checker.ipynb → Main notebook  
- result.txt → Output file
